# 🩺 Medical Specialty Prediction using Machine Learning

A Machine Learning and NLP-based web application that predicts the **medical specialty** from a doctor's medical transcription. The application uses **TF-IDF Vectorization** and a **Multinomial Naive Bayes** classifier to automatically classify clinical text into the appropriate medical specialty.

## 📊 Dataset
- **Dataset:** Medical Transcriptions (MTSamples)
- **Source:** Kaggle
- **Target Column:** `medical_specialty`
